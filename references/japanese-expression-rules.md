# 日本語表現ルール（LP臭を消す）

実LP8本から「効いている表現」と「ありきたりで避けるべき表現」を抜き出して作った。LPらしい誇大・常套句を避け、具体・口語・人間味で書く。

## 目次

- 見出し
- 文末
- 巧い表現の作り方（7原則）
- 禁止・要注意表現

## 見出し

見出しは説明ではなく、状態か心の声を描写する。問いかけは使いすぎない。

良い見出し（実データ）:

- ChatGPTの時代、終わりました。
- 2〜3時間かかってた資料が、10分で終わる。
- それは才能でも努力不足でもありません。
- 準備する壁さえ越えれば、あとは話しかけるだけ。
- でも、それはもう過去の話です。

避ける見出し:

- なぜあなたは活用できていないのか
- 今回のセミナーで学べる内容について
- ◯◯とは何かを徹底解説
- ◯◯が重要な理由

文字量: H2 10〜24字 / H3 8〜22字 / FVメイン16〜34字（2〜4行に分ける）/ バッジ4〜10字を3個まで。

## 文末

丁寧語だけで均さない。短い断定・体言止め・口語・方言を混ぜると台本臭が消える。

使える文末: です / できます / 大丈夫です / もう迷わなくていい / ここから始められます / 次元が違う。/ 終わりました。/ 過去の話です

避ける文末: 重要です / 効果的です / 必要があります / 実現できます / 促進します / 最適化します / と言えるでしょう / 期待できます

1段落は1〜3文。1文は20〜55字。説明が長くなるなら箇条書きかカードに割る。

## 巧い表現の作り方（7原則）

実LPで効いていた表現は、この7つのどれかに当てはまる。テンプレを書く時はこれを使う。

### 1. 抽象語を「具体的な数字・場面」に置き換える
- ✗「短時間で成果が出せる」→ ✓「2〜3時間かかってた資料が、10分で終わる。」
- ベネフィットは時間差・金額・回数で見せる。「便利」「効率化」では何も伝わらない

### 2. 口語・方言で広告臭を消す
- 「ぶっちゃけ次元が違う。」「報われへん…」「AIでええやん。」
- 受講生の声に「（笑）」や素人っぽい読点「2年早く知りたかった、、、」を残すと作り物臭が消える
- 崩しのトーンはターゲット属性に合わせる。主婦・女性層は感情寄りの崩し（「、、、」「もう最高です」）、30〜40代男性会社員は軽い苦笑い系（「（笑）」「結局1つも仕事減ってなかったんですよね」）が自然。属性に合わない崩しは逆に浮くので、ブリーフのターゲットに寄せる

### 3. 型反転で自己否定を解除する
- 「それは才能や努力不足ではなく、再現性の高い◯◯の型を知らないだけ。」
- 読者の挫折を「能力の問題」から外す。購入抵抗を最も下げる定番

### 4. 通説を一度肯定してから反転する
- 「◯◯は稼げない。一度は耳にしたことがあるはずです。でも、それはもう過去の話です。」
- 否定から入らず、納得感を持って新提案へ運ぶ

### 5. プリバンキング（読者の警戒を先に言う）
- 「私も『怪しい』『本当か？』と思うタイプでした。」
- 売り手が先に警戒心を言語化すると、読者の防御が解ける

### 6. 損失回避に寄せる（押し付けない）
- 「間違った学習法のせいでまた挫折してしまう…なんてもったいないですよね…」
- 「あなたもできる！」と煽るより、「もったいない」の感情の方が押し付け感がない

### 7. 余韻・二者択一で締める
- 「参加した瞬間から、あなたも使いこなす側です。」
- 「使いこなすか、奪われるか。」
- 行動後の景色を1文で見せる。読者に想像の余白を残す

## 禁止・要注意表現

### ゼロ許容（出力前に必ず検索して消す）
- これにより / 近年 / さらに / 加えて / まとめると / 今後の展開
- 重要です / 効果的です / 本質的な / 最適な / 活用する / 促進する / 実現する / と言えるでしょう
- em dash（—）・全角ダッシュ・装飾的な長い横線（humanize-ja のゼロ許容に合わせる）

境界メモ: 禁止は「でしょう」「と言えるでしょう」。口語の「でしょ」「でしょ？」は会話調としてOK（型Bの自己開示などで使える）。grepは `でしょう` で検索し、`でしょ。`/`でしょ？` は誤検出しない。

### LP臭い常套句（使わない。実LPにもあったが弱い行）
- 「豪華」◯大特典 → 「豪華」は埋め草。特典の中身で語る
- 「完全」無料/攻略/公開/自動化 → 「完全」の多用は誇大に映る
- 「最短最速」「市場価値の高い」「新時代の」 → 具体ゼロの煽り語
- 「ワクワクする未来」 → 感情語の押し付け。読者の想像の余白を奪う
- 「◯◯が重要です/大切です」 → 当たり前で読み飛ばされる
- 「あなたも◯◯できる！」の連呼 → 1回なら効く。反復するとテンプレ感が前に出る
- 「ぐんぐん」「どんどん」等の擬態語連発 → 盛って見える

### 創作禁止
- 入力にない 実績・人数・満足度・成功率・No.1・受賞歴・メディア掲載 を作らない
- 数字は入力にある根拠のみ。無ければ数字を使わずに書く

## 太字とダッシュ

- Markdownの太字（`**`）は原則使わない。強調は見出し分割と改行で表現する
- em dash・全角ダッシュは使わない。区切りは読点・改行・短文で作る

## 使える型

FV（型ごとに差し替え）:

```
特典型:  [理想の変化]を叶える / [ウェビナータイトル] / 参加者限定 ◯大特典プレゼント
共感型:  [テーマ]って、なんだか難しそう… / そう思っているあなたへ
物語型:  [どん底の状態]だった私が / [ある気づき]で[強い実績]を出した方法を公開
```

悩み（吹き出し）:

```
- [具体的な生活場面]なのに、[うまくいかない事実]
- [周りと比べて焦る心の声]
- [何度も試したのに挫折した事実]
```

型反転:

```
それは、才能や努力不足ではありません。
[正しい順番/型/道具]を知らなかっただけです。
```

CTA:

```
ボタン: 今すぐLINE追加して 勉強会に無料参加する
直前文: まずは気軽にLINE登録。勉強会は参加費0円です。
注記: ※LINEが開かない場合、無料勉強会は終了しています。
```

クロージング:

```
[このまま様子見を続ける損失]。
でも、ここから変えられます。
まずはLINEで、無料参加枠を受け取ってください。
```
