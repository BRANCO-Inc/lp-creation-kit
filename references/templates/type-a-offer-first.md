# 型A: 特典ご利益先出し型（Offer-First）

## この型の性格

「無料でこんなに受け取れる」のお得感で先に手を止めさせる型。
刺さる層は、興味はあるが行動ハードルで止まっている中〜低関心層。悩みを深く抉るより、得られるものの量と無料性を早く見せて、軽い気持ちでLINE登録させる。

実LPの代表: linkup（3大特典）/ realus（6大特典）/ zeromin（5大特典）。

## 書き方の指針

- FVで「理想の変化」と「◯大特典＋無料」を同時に出す。実績数値もFV直下に置く
- 特典は早い段階（FV直後）で物量を見せる。各特典に「網羅性の一言」を付ける（例「全30項目チェック付き」）
- 悩みは深掘りしすぎない。3個まで、軽く当てる程度
- CTAは特典再掲とセットで何度も置く。直前に必ず「参加費0円」を添える
- 「豪華」「完全無料」の連発は避ける。特典の中身と数字で語る（japanese-expression-rules.md 参照）

- [ ] CTAの直前に毎回「参加費0円」がある
- [ ] 「豪華」「完全無料」を連発していない
- [ ] 実績・数値は入力にある根拠のみ
