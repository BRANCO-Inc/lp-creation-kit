# 型B: 悩み共感・型反転型（Empathy → Reframe / PASONA）

## この型の性格

「自分のことだ」と思わせてから、「あなたのせいじゃない」で反転する型。
刺さる層は、問題は自覚しているのに行動できない・過去に独学や挑戦で挫折した経験がある層。
入口で悩みの心の声を代弁し、現状維持の損失を見せ、「才能や努力不足ではなく、正しい型を知らないだけ」で自己否定を解除してから、ウェビナーを必然として提示する。

実LPの代表: opt-in（「インスタってなんか難しそう」→「正しい収益化の型を抑えるだけ」）/ linkup（独学の挫折→正しい学習法）。

## 書き方の指針

- FVは特典を出さない。悩みの代弁か「[テーマ]って、なんだか難しそう…」で入り、小さな希望を添える
- 悩みは吹き出し3〜4連で畳みかける。具体的な生活場面の心の声にする（一般論にしない）
- 型反転（「才能や努力不足ではなく◯◯を知らないだけ」）が山場。1文で言い切る
- 受講生の声は「同じ悩みからの変化」で揃える。ターゲットが自分を重ねられるように
- 説教しない。損失回避（「もったいない」）で寄せる（japanese-expression-rules.md 参照）
