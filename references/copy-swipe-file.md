## 11.「LP臭くない・巧い」と判断した表現と、その理由（分析）
- 「ChatGPTの時代 終わりました」（出典: fusion-ai） → 既存知識を一瞬で陳腐化させ「自分のアップデート不足」を自覚させる、強い言い切りの否定フック。
- 「チャットで使う道具とは ぶっちゃけ次元が違う。」（出典: fusion-ai） → 「ぶっちゃけ」の口語で広告臭を消し、断定で優位性を刷り込む。
- 「2〜3時間かかってた資料が、10分で終わる。」（出典: fusion-ai） → ベネフィットを抽象語でなく「具体的な時間差」で見せ、即イメージできる。
- 「部下に見せたら引いてましたよ（笑）。」（出典: fusion-ai） → 受講生の声に「（笑）」と生活感を入れ、台本臭を消して信憑性を上げる。
- 「「私の価値、時給10円やった…」」（出典: optin-lp1） → 関西弁＋自己卑下で人間味を出し、8億という実績との落差で物語を作る。
- 「頑張ってるのに 報われへん…」（出典: optin-lp1） → 方言の独白で「自分のことだ」と感じさせ、共感の入口を作る。
- 「それは才能や努力不足ではなく〜型を知らないだけ」（出典: optin-lp1 / optin-lp2 / optin-lp3） → 読者の挫折を「能力の問題」から外し、自己否定を解除して購入抵抗を下げる定番の型反転。
- 「なんてもったいないですよね…」（出典: linkup） → 説得でなく「もったいない」という損失回避の感情に寄せ、押しつけ感がない。
- 「そんなあなたに LinkUp」（出典: linkup） → 悩みの羅列からブランドへ自然に橋渡しし、唐突な売り込みにならない。
- 「でも、それはもう 過去の話です」（出典: zeromin） → 通説を一度肯定してから反転させ、納得感を持って新提案へ誘導する。
- 「私も「怪しい」「本当か？」と思うタイプ」（出典: zeromin） → 売り手側が読者の警戒心を先に言語化し、防御を解く（プリバンキング）。
- 「テンプレ通りに融資申請したら 本当に500万円が通った。」（出典: zeromin） → 受講生の声が抽象的な感謝でなく「具体的行動＋具体的金額＋本当に」で再現性を匂わせる。
- 「準備する壁さえ越えれば、 あとは話しかけるだけ。」（出典: fusion-ai） → 難易度を「壁1つ」に矮小化し、行動後の景色まで一文で見せる余韻型クロージング。
- 「2年早く知りたかった、、、ってことばかりで」（出典: realus-lp02a） → 読点の打ち方まで素人っぽく崩し、加工していない生声に見せる。
- 「AIを使いこなすか AIに奪われるか」（出典: fusion-ai） → 二者択一に追い込み、当事者意識と危機感を同時に作る。

## 12. 逆に「ありきたり・LP臭い・避けるべき」と感じた表現
- 「最短最速で身につける」（出典: linkup） → 「最短最速」はあらゆる教育系LPの常套句で、具体性ゼロで記憶に残らない。
- 「豪華6大特典」「豪華100大特典」「豪華5大特典」 → 「豪華」は意味のない強調語。特典の中身で語れない時の埋め草に見える。
- 「完全無料」「完全攻略」「完全公開」「完全自動化」 → 「完全」の多用は誇大に映り、読み手が割り引いて読む。
- 「期間限定で無料開催中」（出典: optin-lp1 等） → ほぼ全LPが使う希少性テンプレで、常時開催が見透かされると逆効果。
- 「収入も時間も自由に設計」「人間関係のストレスゼロ」（出典: realus-lp02a / realus-lp04a） → 比較表の右列が理想を並べすぎて非現実的、かえって信用を落とす。
- 「あなただけの価値が重宝」（出典: realus-lp02a） → 抽象的で何も具体を示さず、聞こえのいいだけの常套句。
- 「市場価値の高い SNSマーケティングスキル」（出典: optin-lp2） → 「市場価値の高い」は中身のない権威付けワードで実感を生まない。
- 「新時代の資産形成」「新時代の民泊運営モデル」（出典: zeromin） → 「新時代」は具体を欠いた煽り語で、何が新しいか伝わらない。
- 「〜が重要です」「〜が大切です」型の説明 → 当たり前で読み飛ばされる。実LPでも吹き出し説明文に散見され、最も弱い行。
- 「ワクワクする未来が描ける」（出典: zeromin） → 感情語の押し付けで、読者が自分で想像する余白を奪っている。
- 「ぐんぐん」「どんどん」等の擬態語連発（出典: linkup） → 体験談を盛って見せる典型で、多用すると作り物臭が出る。
- 「あなたも〜できる！」の連呼 → 主語をあなたにして煽る定番。1回なら効くが反復するとテンプレ感が前面に出る。
