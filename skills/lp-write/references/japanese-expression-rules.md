# 日本語表現ルール（LP臭を消す）

誘導LP本文を書くときの表現の正本。LPらしい誇大・常套句を避け、具体・口語・人間味で書く。読者が「広告を読まされている」と感じた瞬間に離脱するので、台本臭を消すことを最優先にする。

## 見出し

見出しは説明ではなく、状態か心の声を描写する。読者が自分のことだと感じる一文にする。問いかけは効くが使いすぎない（3つ以上連続させない）。

良い見出し（状態・心の声を描く / すべてオリジナルの中立例）:

- 毎晩のスキンケア、もう何が正解か分からなくなってた。
- 教材は3冊買った。続いたのは1冊もなかった。
- それは意志が弱いからじゃありません。
- 口座を開くまでが、いちばん重い。
- でも、そのやり方はもう古い。

避ける見出し（説明・常套句に逃げている）:

- なぜあなたは続けられないのか
- 今回の講座で学べる内容について
- ◯◯とは何かを徹底解説
- ◯◯が重要な理由
- 知らないと損する◯◯の真実

文字量の目安: H2 10〜24字 / H3 8〜22字 / FVメイン16〜34字（2〜4行に分ける）/ バッジ4〜10字を3個まで。長い見出しは2行に割って、読者が一拍で読める形にする。

## 文末

丁寧語だけで均さない。「です・ます」で全部そろえると台本臭が出る。短い断定・体言止め・口語を混ぜると、人が書いた文章に見える。

使える文末（混ぜて使う）:

- です / できます / 大丈夫です
- もう迷わなくていい
- ここから始められます
- 〜だけ。（体言止め）
- 終わりです。/ 過去の話です。
- 〜なんですよね。（軽い口語）

避ける文末（説明調・翻訳調）:

- 重要です / 効果的です
- 必要があります / 期待できます
- 実現できます / 促進します / 最適化します
- と言えるでしょう

1段落は1〜3文。1文は20〜55字。説明が長くなりそうなら、文を割らずに箇条書きかカードに分ける。

## 巧い表現の作り方（7原則）

効くLPコピーは、この7つのどれかに当てはまる。テンプレを書くときはこれを使う。例はすべてオリジナルの中立例で、教育・美容・金融・サービスなどに散らしている。

### 1. 抽象語を「具体的な数字・場面」に置き換える

「便利」「効率化」「成果が出る」では何も伝わらない。ベネフィットは時間差・金額・回数・場面で見せる。

- 避ける: 短い時間で家事がラクになります
- 良い: 30分かかってた献立決め、冷蔵庫を撮るだけで終わる。

数字は入力にある根拠のみ使う（後述の創作禁止）。根拠がなければ場面で具体化する。

### 2. 口語・人間味で広告臭を消す

きれいに整いすぎた文章は、かえって嘘っぽく見える。読者の声には軽い崩しを残す。

- 「正直、半信半疑でした。」
- 「結局、貯金は1円も増えてなかったんですよね。」
- 体験談に素人っぽい読点や言い淀みを残す: 「2年前に知りたかった、、、」「あ、これでよかったんだ、って。」

整いすぎた声より、少し崩れた声のほうが本物に見える。

### 3. 反転で自己否定を解除する

読者の挫折を「能力の問題」から外に出す。購入抵抗をいちばん下げる定番。

- 「続かなかったのは、根性がないからじゃありません。順番が逆だっただけです。」
- 「英語が話せないのは、センスの問題ではなく、練習する場所がなかっただけ。」

「あなたが悪い」を一度も言わずに、原因を外側に置く。

### 4. 通説を一度肯定してから反転する

否定から入らず、読者の前提に一度うなずいてから新提案へ運ぶ。納得感が乗る。

- 「投資は怖い。よく聞く話です。たしかに、昔のやり方ならそうでした。でも今は違います。」
- 「肌は年齢には勝てない。そう思ってました。少し前までは。」

いきなり「それは間違いです」と言わない。一度肯定してから景色を変える。

### 5. プリバンキング（読者の警戒を先に言う）

読者が思っている疑いを、売り手が先に言葉にする。先回りされると防御がゆるむ。

- 「『どうせ高い商品を売られる』と思いますよね。私もそう思って読んでました。」
- 「怪しいと感じてる人ほど、ここまで読んでほしいです。」

警戒を隠さず、先に拾う。型2・型3では2箇所ほど入れると効く。

### 6. 損失回避に寄せる（押し付けない）

「あなたもできる」と煽るより、「もったいない」のほうが押し付け感が出ない。

- 「合わない化粧品を使い続けて、また肌を荒らしてしまう。それ、けっこうもったいないです。」
- 「やり方を知らないまま、もう1年同じ場所で足踏みする。それだけは避けたいですよね。」

恐怖で締め上げず、静かに損を惜しむ温度にする。

### 7. 余韻・二者択一で締める

行動後の景色を1文で見せて、想像の余白を残す。説明で埋めない。

- 「申し込んだ瞬間から、悩む側ではなくなります。」
- 「このまま様子を見るか、今日から変えるか。」

読者に決めさせる余白を残す。全部を言い切らない。

## 禁止・要注意表現

### ゼロ許容（出力前に必ず検索して消す）

接続・締めの常套句:

- これにより / 近年 / さらに / 加えて / まとめると / 今後の展開

説明調の述語:

- 重要です / 効果的です / 本質的な / 最適な / 活用する / 促進する / 実現する / と言えるでしょう

記号:

- em dash（—）・全角ダッシュ・装飾的な長い横線

境界メモ: 禁止は「でしょう」「と言えるでしょう」。口語の「でしょ」「でしょ？」は会話調としてOK（読者の声や自己開示で使える）。検索は `でしょう` で行い、`でしょ。`／`でしょ？` を巻き込まないようにする。

### 創作禁止

入力にない次の情報を作らない。

- 実績・人数・満足度・成功率・No.1・受賞歴・メディア掲載

数字は入力にある根拠のみ。根拠がなければ、数字を使わず場面で具体化する。

## LP常套句

実在のLPによく出るが、効きが弱い・誇大に映る言い回し。使わない。

- 「豪華」◯大特典 → 「豪華」は埋め草。特典の中身そのもので語る
- 「完全」無料／攻略／公開／自動化 → 「完全」の多用は誇大に映る
- 「最短最速」「市場価値の高い」「新時代の」 → 具体ゼロの煽り語
- 「ワクワクする未来」 → 感情語の押し付け。読者の想像の余白を奪う
- 「◯◯が重要です／大切です」 → 当たり前で読み飛ばされる
- 「あなたも◯◯できる！」の連呼 → 1回なら効く。反復するとテンプレ感が前に出る
- 「ぐんぐん」「どんどん」「サクサク」等の擬態語連発 → 盛って見える

## 崩しトーンと読者属性

口語や崩しは、読者属性に合わせて温度を変える。属性に合わない崩しは逆に浮く。ブリーフのターゲットに必ず寄せる。

- 主婦・女性層: 感情寄りの崩し（「、、、」「もう手放せない」「あ、これでよかったんだ」）
- 30〜40代の男性会社員: 軽い苦笑い系（「正直、半信半疑でした」「結局なにも変わってなかったんですよね」）
- 専門職・高単価・BtoB寄り: 崩しは最小限。具体と根拠で信頼を作り、口語は1〜2箇所のアクセントに留める

迷ったら、ターゲットが普段使う言葉を想像して、そこから1段だけ崩す。崩しすぎない。

## 太字とダッシュ

- Markdownの太字（`**`）は使わない。強調は見出しの分割と改行で表す。1行に1つの主張、改行で間を作る、という形で目を引く
- em dash・全角ダッシュは使わない。区切りは読点・改行・短文で作る

## 使える型

すべてオリジナル・中立。プレースホルダは `[ ]` で示す。ジャンルに合わせて差し替える。

FV（型ごとに差し替え）:

```
オファー先出し:  [理想の状態]を、[無料/0円]で。/ [オファー名] / 参加者に ◯つの特典
自己投影・反転:  [テーマ]って、なんだか難しそう…… / そう思っているあなたへ
権威・実証先出し: [第三者の評価/規模] / [変化の証明] を、まず見てください
```

悩み（吹き出し）:

```
- [具体的な生活場面]なのに、[うまくいかない事実]。
- [周りと比べて焦る心の声]。
- [何度も試したのに挫折した事実]。
```

反転:

```
それは、[能力/才能/根性]の問題ではありません。
[正しい順番／道具／環境]が無かっただけです。
```

CTA:

```
ボタン: 今すぐ[登録/予約]して [無料オファー]を受け取る
直前文: まずは気軽に[登録方法]。[費用]は0円です。
注記: ※[リンクが開かない場合]、[オファー]は終了しています。
```

クロージング:

```
[このまま様子見を続けたときの小さな損失]。
でも、ここから変えられます。
まずは[登録方法]で、[無料の枠/特典]を受け取ってください。
```
